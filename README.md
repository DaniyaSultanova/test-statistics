# Interactive Line Chart Dashboard

Interactive Line Chart that visualizes A/B test statistics with multi-line filtering and export functionality. 

## Features

- Multi-line chart visualization using [Recharts](https://recharts.org/)
- Custom select components for filtering lines using [React Select](https://react-select.com/home)
- Day/Week view selection
- Responsive layout

**(Bonus features)**
- Line style toggle: Line, Smooth, Area
- Light/Dark theme toggle
- Export chart as PNG

## Tech Stack

- React 18
- TypeScript
- Recharts
- React Select
- SCSS Modules
- Vite


## Installation

1. **Clone the repository**

   ```sh
   git clone https://github.com/DaniyaSultanova/test-statistics.git
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

3. **Run the Project**

   ```sh
   npm run dev
   ```
